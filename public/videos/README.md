# Videos Directory

This directory contains video files for the VideoShowcase component.

## Required Videos

Please add the following video files to this directory:

1. **luxury-transport.mp4** - Main luxury vehicle transport video
2. **professional-service.mp4** - Professional service demonstration
3. **global-reach.mp4** - International transport services
4. **secure-transport.mp4** - Secure vehicle transportation

## Video Specifications

- **Format**: MP4 (H.264)
- **Resolution**: 1920x1080 (Full HD) recommended
- **Duration**: 30-60 seconds per video
- **File Size**: Optimized for web (under 10MB per video)
- **Aspect Ratio**: 16:9

## Thumbnail Images

The component uses existing images from `/images/` directory as thumbnails:
- c1.jpg (for luxury-transport.mp4)
- c2.jpg (for professional-service.mp4)
- c3.jpg (for global-reach.mp4)
- c4.jpg (for secure-transport.mp4)

## Adding Your Videos

1. Place your video files in this directory
2. Ensure they match the filenames above
3. Test the video showcase on your website
4. Adjust video titles and descriptions in `VideoShowcase.tsx` if needed

## Tips for Great Videos

- Use high-quality footage of your services
- Include your branding and logo
- Show different types of vehicles you transport
- Highlight your professional team
- Keep videos engaging and under 60 seconds
- Add background music or voiceover if desired
